# About Time Freezer
This mod is super simple—it just removes the day-night cycle from the game. But if you want to play at night, I recommend installing and enabling this mod. It’s also useful for creating a horror survival game on Luanti, for example, even though I have to admit that horror mods are more common on Minecraft Java than on Luanti.
